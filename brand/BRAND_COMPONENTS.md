# UI Component Guidelines

## Buttons

- Shape: 4px radius  
- Primary: Black background, white text  
- Secondary: White background, black text  
- Hover: Subtle opacity change  
- No heavy shadows  

## Cards

- Minimal shadows  
- 1px border (#E0E0E0)  
- Large image area  
- Clean typography  
- Consistent padding  

## Forms

- Full‑width fields  
- 12px vertical spacing  
- Clear labels (no placeholder‑only fields)  
- Minimal borders  

## Navigation

- Simple top bar  
- No dropdown clutter  
- Clear CTAs: “Shop”, “Create”, “Collections”
